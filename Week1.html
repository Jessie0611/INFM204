Complete the following two questions for this assignment. To help you with this assignment, a demo will be given in class with the recording posted after class on the Lecture Videos page of this week's module.

  1. Draw a flowchart to represent the logic of a program that allows the user to enter two values. The program outputs the sum of and the difference between the two values.
  2. Write pseudocode to represent the logic of a program that allows the user to enter a value for hours worked in a day. The program calculates the hours worked in a five-day week and the hours worked in a 252-day work year. The program outputs all the results.
